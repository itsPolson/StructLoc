# Contributing to the project

Thank you for your interest in contributing to StructLoc! We welcome contributions from the community and appreciate any help in improving the plugin. Here are some guidelines to help you get started:
- Open a pull request with a clear description of the changes you have made and why they are necessary.
- Make detailed bug reports with steps to reproduce the issue, expected behavior, and actual behavior.
- Help improve the documentation by adding new pages or updating existing ones with accurate and helpful information.
