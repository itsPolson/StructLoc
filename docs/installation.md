# Installation

This plugin is compatible with Paper/Bukkit up to version `1.21.11`.

To install the plugin, follow these steps:

1. Download the latest `StructLoc.jar` file from the [GitHub releases](https://github.com/StructLoc/StructLoc/releases) page.
2. Place the `StructLoc.jar` file in your server's `plugins` directory.
3. Start or restart your server to load the plugin.
