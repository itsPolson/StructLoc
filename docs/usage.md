# Usage

| Command                     | Description                                                    | Permissions       |
| --------------------------- | -------------------------------------------------------------- | ----------------- |
| `/structloc <structure_ID>` | Displays the location of the specified structure in the world. | `structloc.use`   |
| `/structloc reload`         | Reloads the plugin's configuration files.                      | `structloc.admin` |
| `/givestructloc <player>`   | Gives the specified player a StructLoc item.                   | `structloc.admin` |
